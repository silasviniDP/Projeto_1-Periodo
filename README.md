# Sistema de Venda de Ingressos de Cinema

**Status:** Em desenvolvimento  

##  Descrição
Sistema desenvolvido em linguagem C para simular a venda de ingressos de cinema.  
Permite visualizar filmes, escolher sessões, selecionar assentos e finalizar compras.

##  Autores

## Código Principal
O projeto foi desenvolvido na linguagem C.

## Funcionalidades
- Exibir filmes disponíveis
- Menu de navegação
- Cadastro de sessões
- Seleção de assentos
- Carrinho de compras
- Finalização de pedido

##  Arquivos Gerados
- `filmes.txt` → lista de filmes  
- `sessoes.txt` → horários  
- `ingressos.txt` → compras realizadas  
- `assentos.txt` → mapa de assentos
